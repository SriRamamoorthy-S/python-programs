# Voice-Controlled Home Automation Project

## Project Details
This project aims to create a comprehensive home automation system that can be controlled using voice commands. The system integrates various home devices and allows users to control them seamlessly through voice recognition technology.

## Components
- **Microcontroller**: Arduino/Raspberry Pi
- **Voice Recognition Module**: Google Assistant, Alexa or custom system using libraries such as SpeechRecognition and pyttsx3.
- **Home Appliances Control**: Smart Lights, Smart Plugs, Thermostats, Cameras, etc.
- **Communication Protocols**: MQTT, HTTP REST API for controls.
- **Software Libraries**: `speech_recognition`, `pyttsx3`, `paho-mqtt`, etc.

## Setup Instructions
1. **Prerequisites**:
   - Make sure you have Python installed (preferably version 3.6 or higher).
   - Install required libraries using pip:
   ```bash
   pip install speech_recognition pyttsx3 paho-mqtt
   ```

2. **Hardware Setup**:
   - Connect the microcontroller to your computer and set up home devices with the appropriate modules.
   - Ensure the voice recognition module can interface with your selected microcontroller.

3. **Code Setup**:
   - Clone the repository:
   ```bash
   git clone https://github.com/SriRamamoorthy-S/python-programs.git
   cd python-programs
   ```
   - Run the main program:
   ```bash
   python main.py
   ```

4. **Testing**:
   - Speak the commands to control your home devices (e.g., "Turn on the lights", "Set the thermostat to 72 degrees").
